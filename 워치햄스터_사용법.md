# 🐹 워치햄스터 통합 관리 센터

## 🚀 사용법

### 실행
```bash
🐹워치햄스터_통합_관리_센터.bat
```

### 메뉴
- **1**: POSCO 뉴스 모니터링 시작
- **A**: 전체 시스템 상태 확인
- **B**: 전체 시스템 업데이트
- **C**: 통합 로그 관리
- **D**: 전체 시스템 테스트
- **E**: 전체 백업 생성
- **F**: 시스템 설정
- **0**: 종료

## 🔧 문제 해결

### ANSI 색상 코드가 보이는 경우
1. Windows Terminal 사용 권장
2. 또는 기존 버전 사용: `🐹워치햄스터_총괄_관리_센터.bat`

### 파일을 찾을 수 없는 경우
- 올바른 디렉토리에서 실행했는지 확인
- `Monitoring\Posco_News_mini` 폴더 존재 확인
