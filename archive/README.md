# ARCHIVE 디렉토리

## 개요
완료된 작업 및 백업 파일들

## 목적
작업 완료 보고서, 마이그레이션 로그, 백업 파일 보관

## 구조

### 하위 디렉토리

- **task_summaries/**: 작업 완료 보고서들
- **migration_logs/**: 마이그레이션 로그
- **backups/**: 백업 파일들
- **temp/**: 임시 파일들

## 사용 가이드

### 파일 추가 시 주의사항
- 이 디렉토리의 목적에 맞는 파일만 추가하세요
- 파일명은 명확하고 일관된 네이밍 규칙을 따르세요
- 중요한 파일은 백업을 생성하세요
