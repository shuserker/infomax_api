## 🔧 **초기 설정 (최초 1회만)**

### **⚠️ 한글 깨짐 문제 해결**

**🎯 왜 .ps1 파일을 더블클릭하면 PowerShell로 실행되지 않나요?**
- Windows 보안 정책상 PowerShell 스크립트는 더블클릭으로 실행되지 않습니다
- 실수로 악성 스크립트가 실행되는 것을 방지하기 위함입니다

**✅ 해결 방법 (가장 쉬운 순서대로):**

**방법 1: 배치 파일 사용 (가장 쉬움, 권장)**
```batch
# 더블클릭으로 바로 실행 가능!
🚀POSCO뉴스_시작.bat          # 워치햄스터 시작
⏹️POSCO뉴스_중지.bat          # 워치햄스터 중지  
📊로그확인.bat               # 로그 확인
🔧환경설정.bat               # 환경 설정
```

**방법 2: 관리자 권한 필요한 작업**
```batch
# 더블클릭으로 바로 실행 (자동으로 관리자 권한 요청)
🔧PowerShell실행정책설정_관리자권한.bat
🔧환경설정_관리자권한.bat
```

**방법 3: PowerShell 메뉴 실행기**
```powershell
# 우클릭 → PowerShell로 실행
🔧한글깨짐_해결_실행.ps1
```

### **1단계: PowerShell 실행 정책 설정**
```batch
# 더블클릭으로 실행 (자동으로 관리자 권한 요청)
```

## 🖥️ **Windows 작업 스케줄러 설정**

### **PowerShell 스크립트 등록**
1. `Win + R` → `taskschd.msc` 입력
2. **"기본 작업 만들기"** 클릭
3. **이름**: `POSCO 워치햄스터 🛡️ 자동 시작`
4. **트리거**: `컴퓨터를 시작할 때`
5. **동작**: `프로그램 시작`
6. **프로그램**: `PowerShell.exe`
7. **인수**: `-File "D:\path\to\🔧POSCO뉴스_서비스_시작.ps1"`
8. **설정**: 
   - ✅ `사용자가 로그온하지 않아도 실행`
   - ✅ `가장 높은 권한으로 실행`
