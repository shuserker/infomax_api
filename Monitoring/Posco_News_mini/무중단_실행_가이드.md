# POSCO 뉴스 모니터 - 무중단 실행 가이드

## 🚀 Windows 작업 스케줄러 설정 방법

### 1단계: 작업 스케줄러 열기
- `Win + R` → `taskschd.msc` 입력 → 엔터

### 2단계: 기본 작업 만들기
1. 우측 패널에서 **"기본 작업 만들기"** 클릭
2. **이름**: `POSCO 뉴스 모니터`
3. **설명**: `POSCO 뉴스 자동 모니터링 시스템`

### 3단계: 트리거 설정
- **"컴퓨터를 시작할 때"** 선택

### 4단계: 동작 설정
- **"프로그램 시작"** 선택
- **프로그램/스크립트**: 
  ```
  D:\DEV\GIT_DEV\infomax_api\infomax_api\Monitoring\Posco_News_mini\start_posco_monitor.bat
  ```
- **시작 위치**: 
  ```
  D:\DEV\GIT_DEV\infomax_api\infomax_api\Monitoring\Posco_News_mini
  ```

### 5단계: 추가 설정
- **"사용자가 로그온하지 않아도 실행"** 체크
- **"가장 높은 권한으로 실행"** 체크

## 📋 **사용법**

### 즉시 시작
```bash
# 배치 파일 더블클릭
start_posco_monitor.bat
```

### 로그 확인
```bash
# 로그 보기
view_monitor_log.bat
```

### 중지
```bash
# 모니터링 중지
stop_posco_monitor.bat
```

## ✅ **장점**
- 🔄 **키로 업데이트와 무관**: 키로 재시작해도 계속 실행
- 🚀 **시스템 부팅 시 자동 시작**: 컴퓨터 켜면 자동 실행
- 📊 **로그 파일 저장**: `monitor.log`에서 모든 기록 확인
- 🛑 **언제든 중지/재시작 가능**: 배치 파일로 간편 제어
